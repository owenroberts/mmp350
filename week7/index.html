Calculating Color

MMP 350 > Week 7

We’re going to look at color today and some cool stuff that SASS can do for us using new color concepts. First let’s review some basic color theory so we have a basis for how to use SASS.

We've all looked at the Adobe Kuler site for making color schemes. But what do the different Color Rules (analogous, monochromatic, triad, complementary) signify? These are basic concepts in color theory. Let's take a look.

The main thing I want to take away from these concepts is the fact that color relationships can be described with numbers. Because all colors exist on the color wheel, we can set up basic relationships, for example complementary colors are separated by 180 degrees, and apply them to any specific color.

Sass uses HSL functions making it easy to set up these color rules in a way that we can substitute any color and achieve the same basic scheme (or our user can choose a color). This site shows us how these rules work: SassMe.

Let's choose a color. Now let's take a look at how to create these color schemes with simple SCSS code.

Sass color functions:

  • adjust_hue(color, degrees) change the hue of a color by degrees number 0 - 360
  • darken(color, precent) darker color 0% - 100%
  • lighten(color, precent) lighter color 0% - 100%
  • saturate(color, precent) more saturated color 0% - 100%
  • desaturate(color, precent) less saturated color 0% - 100%
