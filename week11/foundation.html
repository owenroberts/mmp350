WordPress theme: Adding foundation and styles to front-page.php

MMP 350 > Week 11

Today we'll begin adding the Foundation classes and styling from the portfolio we prototyped in HTML and CSS to our php template pages.

This process involves using both the WordPress dashboard to add and modify content, as well as our template/theme files.

Let's start by adding foundation.css to the theme files and and adding some classes into front-page.php

Add the foundation stylesheet into header.php:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Theme Starter Files</title>
    <link rel="stylesheet" href="foundation.min.css" type="text/css" />
    <link rel="stylesheet" href="wp-content/themes/mytheme/style.css">
</head>

Since we're in header.php let's also make the <header> section show-for-small-only. You may not need this for your front page, but mine doesn't show the header section in the desktop version:

<header class="show-for-small-only">

Now, moving to front-page.php we can add the class="row medium-up-3" used to create the front page grid:

<div class="posts row medium-up-3" id="front-page">

And each article will now become a columns and we can also use the project class to get our CSS styles from the template:

<article class="column project" id="<?php the_ID(); ?>">

From here, we can add and continue to modify foundation classes to recreate the rest of the template.
